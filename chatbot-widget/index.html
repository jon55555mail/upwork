One-Line Embed

Add this single line to your website's HTML to deploy the chatbot instantly:

<script src="https://cdn.chatflow.ai/widget.js" data-bot="acme-123" data-color="#6c63ff"></script>

Advanced Configuration

Full control over widget behavior, appearance, and AI model:

<script> ChatFlow.init({ botId: 'acme-123', model: 'claude-sonnet', color: '#6c63ff', position: 'bottom-right', greeting: 'Hi! How can I help?', quickReplies: ['Services', 'Pricing', 'Support'], personality: 'friendly customer support agent', memory: true, analytics: true }); </script>
